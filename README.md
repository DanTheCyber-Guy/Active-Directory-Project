# Active-Directory-Project
<h2>Active Directory Creation</h2>



<h2>Description</h2>
In this tutorial, we're going to walk through how to create an Active Directory hom elab enviornemtn using Oracle Virtual Box. Configuring and running this lab will definately help develop your understanding of how an active directory and windows networking works. So I highly recommend reading through the whole tutorial and view all the picture before trying to complete this yourself.
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>Oracle Virtual Box</b>

<h2>Environments Used </h2>

- <b>Windows 10</b>
- <b>Windows Server 19</b>
